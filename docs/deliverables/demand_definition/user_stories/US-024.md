# US-024: 低在庫の商品を把握する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品管理業務(管理者向け)

**As a** 管理者
**I want** 商品ごとに低在庫のしきい値を設定し、在庫が少なくなった商品を管理画面で一目で把握したい
**So that** 欠品による販売機会損失に気づき、早めに補充の判断ができる

**関連する業務フロー図ステップ**: [[../business_flow/06_product_admin|06_product_admin.md]]「業務フロー図(To-Be): 低在庫アラート確認」

**複雑度判定**: 単純(しきい値保存・在庫比較による絞り込み・表示のみで、状態遷移や外部サービス連携を伴わないため、ユースケース化せず機能一覧に直接反映する)

**Confirmation(受け入れ条件)**:

```
Given 管理者がログイン済みである
When 商品管理画面で商品の低在庫しきい値を設定する
Then しきい値が商品に保存される

Given 管理者がログイン済みであり、しきい値が設定済みで在庫がしきい値以下の商品が存在する
When 商品管理画面またはダッシュボードを開く
Then 該当商品が低在庫として一覧・バッジ表示される

Given しきい値が未設定(NULL)の商品が存在する
When 商品管理画面またはダッシュボードを開く
Then 在庫数によらず当該商品は低在庫として表示されない
```
