# US-011: デフォルトの配送先を設定する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 配送先管理業務

**As a** 顧客
**I want** 複数登録した配送先の中から、よく使う1件をデフォルトに設定したい
**So that** 注文のたびに配送先を選び直さずに済む

**関連する業務フロー図ステップ**: 配送先管理業務(登録フローと同様のロジックを、既存の配送先に対して適用する)

**複雑度判定**: 単純(分岐条件0: 対象の配送先が存在しない場合のみ404エラー)

**Confirmation(受け入れ条件)**:

```
Given 顧客が配送先を2件登録している
When そのうち1件を「デフォルトに設定」する
Then その配送先がデフォルトになり、もう1件のデフォルト指定は解除される
```
