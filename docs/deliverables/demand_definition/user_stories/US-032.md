# US-032: 配送先を確認・編集する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 配送先管理業務

**As a** ログイン済み顧客

**I want** 登録済みの配送先を一覧で確認し、内容を編集したい

**So that** 注文時に正しい配送先を選べる

**関連する業務フロー図ステップ**: [[../business_flow/05_address|05_address.md]]「業務フロー図(To-Be)」

**複雑度判定**: 単純(本人所有の配送先の参照・編集と、デフォルト指定時の一意化で完結するため)

**Confirmation(受け入れ条件)**:

```text
Given 顧客がログイン済みで、配送先が登録されている
When 顧客が配送先一覧を開く
Then 本人の配送先だけがデフォルト優先、同順位では作成日時の新しい順に表示される

Given 顧客がログイン済みで、本人の配送先を編集する
When 有効な配送先情報を保存する
Then 指定した項目が更新され、デフォルト指定した場合は他の配送先のデフォルト指定が解除される

Given 存在しない配送先、または他の顧客の配送先IDが指定される
When 顧客が配送先を編集する
Then 配送先が存在しないことを示すエラーが返る
```
