# US-015: 商品画像を管理する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品管理業務(管理者向け)

**As a** 管理者
**I want** 商品に複数の画像を追加・編集・削除したい
**So that** 顧客が商品を多角的に確認できる

**関連する業務フロー図ステップ**: 商品管理業務 To-Be「商品画像を追加する」

**複雑度判定**: 単純(分岐条件1つ: 対象商品が存在しない)

**Confirmation(受け入れ条件)**:

```
Given 管理者が既存商品に画像URLを追加する
When 追加操作を行う
Then その商品の画像一覧に追加される
```
