# US-030: 関連商品を確認する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品閲覧業務

**As a** 顧客

**I want** 閲覧中の商品と関連する別の商品を確認したい

**So that** 類似する購入候補を比較できる

**関連する業務フロー図ステップ**: [[../business_flow/01_product_purchase|01_product_purchase.md]]「業務フロー図(To-Be)」の商品閲覧ステップ

**複雑度判定**: 単純(同一カテゴリを優先した最大4件の参照で完結するため)

**Confirmation(受け入れ条件)**:

```text
Given 閲覧中の商品と同じカテゴリの別商品が存在する
When 顧客が商品詳細を開く
Then 閲覧中の商品を除く同一カテゴリの商品が最大4件表示される

Given 閲覧中の商品にカテゴリが設定されていない
When 顧客が商品詳細を開く
Then 閲覧中の商品を除く商品から最大4件表示される

Given 存在しない商品IDが指定される
When 顧客が関連商品を取得する
Then 商品が存在しないことを示すエラーが返る
```
