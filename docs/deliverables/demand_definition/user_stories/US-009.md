# US-009: 商品にレビューを投稿する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: レビュー投稿業務

**As a** 顧客
**I want** 商品に評価とコメントを投稿したい
**So that** 自分の購入体験を共有し、他の顧客の購入判断の参考にしてもらえる

**関連する業務フロー図ステップ**: レビュー投稿業務 To-Be「商品詳細画面で評価・コメントを入力し投稿する」

**複雑度判定**: 複雑(分岐条件3つ: 商品が存在しない, 評価が1〜5の範囲外, その商品に投稿済み)
**該当ユースケース**: UC-004(`01_use_cases.md`)

**Confirmation(受け入れ条件)**: 分岐が複雑なため、ユースケースUC-004側に記載する。
