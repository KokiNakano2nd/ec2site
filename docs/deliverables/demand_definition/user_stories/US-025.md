# US-025: クーポンの残数僅少を把握する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: クーポン管理業務(管理者向け)

**As a** 管理者
**I want** クーポンごとに残数僅少のしきい値を設定し、使用回数の残りが少なくなったクーポンを管理画面で一目で把握したい
**So that** クーポン不正利用の兆候や、意図せぬ枯渇に早めに気づける

**関連する業務フロー図ステップ**: [[../business_flow/07_coupon_admin|07_coupon_admin.md]]「業務フロー図(To-Be): クーポン残数アラート確認」

**複雑度判定**: 単純(しきい値保存・残数比較による絞り込み・表示のみで、状態遷移や外部サービス連携を伴わないため、ユースケース化せず機能一覧に直接反映する)

**Confirmation(受け入れ条件)**:

```
Given 管理者がログイン済みである
When クーポン管理画面でクーポンの残数アラートしきい値を設定する
Then しきい値がクーポンに保存される

Given 管理者がログイン済みであり、しきい値が設定済みで残り使用回数(max_uses - used_count)がしきい値以下のクーポンが存在する
When クーポン管理画面またはダッシュボードを開く
Then 該当クーポンが残数僅少として一覧・バッジ表示される

Given しきい値が未設定(NULL)のクーポン、または使用回数上限(max_uses)が無制限(NULL)のクーポンが存在する
When クーポン管理画面またはダッシュボードを開く
Then 残り使用回数によらず当該クーポンは残数僅少として表示されない
```
