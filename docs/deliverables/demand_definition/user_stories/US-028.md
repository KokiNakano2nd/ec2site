# US-028: 商品を検索・閲覧する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品閲覧業務

**As a** 顧客

**I want** 商品を一覧・キーワード検索し、選んだ商品の詳細を確認したい

**So that** 購入候補の商品を見つけ、内容を理解した上で購入を判断できる

**関連する業務フロー図ステップ**: [[../business_flow/01_product_purchase|01_product_purchase.md]]「業務フロー図(To-Be)」の商品閲覧ステップ

**複雑度判定**: 単純(一覧、任意キーワードによる部分一致検索、詳細参照で完結するため)

**Confirmation(受け入れ条件)**:

```text
Given 商品が登録されている
When 顧客が商品一覧を開く、または商品名・説明に含まれるキーワードで検索する
Then 条件に一致する商品が一覧表示される

Given 商品一覧に対象商品が存在する
When 顧客が商品を選択する
Then 商品名・説明・価格・在庫等の商品詳細が表示される

Given 存在しない商品IDが指定される
When 顧客が商品詳細を取得する
Then 商品が存在しないことを示すエラーが返る
```
