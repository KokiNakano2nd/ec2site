# US-034: 注文一覧を確認する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 注文管理業務(管理者向け)

**As a** 管理者

**I want** 全顧客の注文を一覧で確認したい

**So that** 対応が必要な注文を把握し、状態更新や返品判断を行える

**関連する業務フロー図ステップ**: [[../business_flow/08_order_admin|08_order_admin.md]]「業務フロー図(To-Be)」

**複雑度判定**: 単純(管理者による作成日時降順の参照のみで状態変更を伴わないため)

**Confirmation(受け入れ条件)**:

```text
Given 管理者がログイン済みで、注文が存在する
When 管理者が注文管理を開く
Then 全顧客の注文が作成日時の新しい順に表示される

Given 一般顧客がログイン済み、または未認証である
When 管理者向け注文一覧の取得を試みる
Then 権限不足または認証失敗として拒否される
```
