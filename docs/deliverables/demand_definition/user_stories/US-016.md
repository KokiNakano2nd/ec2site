# US-016: クーポンを発行する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: クーポン管理業務(管理者向け)

**As a** 管理者
**I want** 割引クーポンを発行したい
**So that** 販促キャンペーンを実施できる

**関連する業務フロー図ステップ**: クーポン管理業務 To-Be「クーポンコード・割引種別・割引値・使用回数上限を入力し発行する」

**複雑度判定**: 単純(分岐条件2つ: 割引タイプが不正, クーポンコードの重複)

**Confirmation(受け入れ条件)**:

```
Given 管理者が新しいクーポンコード・割引タイプ(定率/定額)・割引値を入力する
When 発行する
Then クーポンが有効な状態で登録される

Given 既に存在するクーポンコードを指定する
When 発行しようとする
Then 「このクーポンコードはすでに存在します」というエラーが表示される
```
