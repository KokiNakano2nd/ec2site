# US-001: カートに商品を追加する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品購入業務

**As a** 顧客
**I want** 気に入った商品をカートに追加したい
**So that** まとめて注文手続きができる

**関連する業務フロー図ステップ**: 商品購入業務 To-Be「商品をカートに追加する」

**複雑度判定**: 単純(分岐条件1つ: 在庫超過)

**Confirmation(受け入れ条件)**:

```
Given 商品の在庫数が5個である
When 顧客がその商品を数量3でカートに追加する
Then カートに数量3で商品が追加される

Given 商品の在庫数が5個である
When 顧客がその商品を数量6でカートに追加する
Then 「在庫数を超える数量は指定できません」というエラーが表示される
```
