# US-022: 返品を申請する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品購入業務

**As a** 顧客
**I want** 発送済みの注文について返品を申請したい
**So that** 商品到着後に不良品や誤発送等の問題があった場合、返金を受けられる

**関連する業務フロー図ステップ**: [[../business_flow/01_product_purchase|01_product_purchase.md]]「業務フロー図(To-Be): 注文キャンセル・返品申請」

**複雑度判定**: 複雑(分岐条件1: 対象注文のステータスが発送済み以外の場合。承認は管理者側(US-023)に委ねられ即時確定しないため、ユースケース化(UC-007)する)

**Confirmation(受け入れ条件)**:

```
Given 顧客がログイン済みであり、自分の注文(ステータス: 発送済み)が存在する
When 注文履歴画面で当該注文の返品理由を入力し「返品を申請する」を選択する
Then 注文が返品申請中になり、管理者の承認待ちとなる(この時点では在庫・返金は変更されない)

Given 顧客がログイン済みであり、自分の注文(ステータス: 受付中/処理中/返品申請中/返品完了/キャンセル済み)が存在する
When 注文履歴画面で当該注文の「返品を申請する」を選択する
Then 「この注文は返品を申請できません」エラーが表示され、注文は変更されない
```
