# US-012: 配送先を削除する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 配送先管理業務

**As a** 顧客
**I want** 不要になった配送先を削除したい
**So that** 配送先一覧を整理できる

**関連する業務フロー図ステップ**: 配送先管理業務(登録フローと同様のロジックを、既存の配送先に対して適用する)

**複雑度判定**: 単純(分岐条件0: 対象の配送先が存在しない場合のみ404エラー)

**Confirmation(受け入れ条件)**:

```
Given 顧客が配送先を1件登録している
When その配送先を削除する
Then 配送先一覧からその配送先が消える
```
