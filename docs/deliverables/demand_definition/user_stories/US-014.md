# US-014: 商品情報を編集・削除する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品管理業務(管理者向け)

**As a** 管理者
**I want** 既存商品の情報を編集・削除したい
**So that** 価格改定や販売終了に対応できる

**関連する業務フロー図ステップ**: 商品管理業務(登録フローと同様のロジックを、既存の商品に対して適用する)

**複雑度判定**: 単純(分岐条件1つ: 対象商品が存在しない)

**Confirmation(受け入れ条件)**:

```
Given 管理者が既存の商品を編集する
When 価格・在庫数等を変更して保存する
Then 商品情報が更新される

Given 存在しない商品IDに対して編集・削除を行おうとする
When 操作を実行する
Then 「商品が見つかりません」というエラーが表示される
```
