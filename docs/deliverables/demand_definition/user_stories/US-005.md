# US-005: 注文確定後に確認メールを受け取る

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品購入業務

**As a** 顧客
**I want** 注文が確定したら確認メールを受け取りたい
**So that** 注文内容が正しく登録されたことを確認できる

**関連する業務フロー図ステップ**: 商品購入業務 To-Be「注文確認メールを送信する」

**複雑度判定**: 単純(分岐条件0: 注文確定時に必ず送信)

**Confirmation(受け入れ条件)**:

```
Given 顧客が注文を確定した(決済が成功した)
When 注文が確定処理される
Then 顧客のメールアドレス宛に、注文番号・購入商品・金額を含む確認メールが送信される
```
