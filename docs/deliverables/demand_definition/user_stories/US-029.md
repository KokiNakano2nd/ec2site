# US-029: 商品の複数画像を確認する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品閲覧業務

**As a** 顧客

**I want** 商品に登録された複数の画像を表示順に確認したい

**So that** 商品の外観を複数の観点から把握して購入を判断できる

**関連する業務フロー図ステップ**: [[../business_flow/01_product_purchase|01_product_purchase.md]]「業務フロー図(To-Be)」の商品閲覧ステップ

**複雑度判定**: 単純(対象商品の画像を表示順に参照するだけで状態変更を伴わないため)

**Confirmation(受け入れ条件)**:

```text
Given 商品に複数の追加画像が登録されている
When 顧客が商品詳細を開く
Then 追加画像が管理者の指定した表示順で表示される

Given 存在しない商品IDが指定される
When 顧客が画像一覧を取得する
Then 商品が存在しないことを示すエラーが返る
```
