# US-018: 注文ステータスを更新する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 注文管理業務(管理者向け)

**As a** 管理者
**I want** 注文の処理状況(受付中/処理中/発送済み/完了)を更新したい
**So that** 出荷状況を管理し、顧客に進捗を知らせられる

**関連する業務フロー図ステップ**: 注文管理業務 To-Be「対象の注文のステータスを更新する」

**複雑度判定**: 単純(分岐条件1つ: 対象の注文が存在しない)

**Confirmation(受け入れ条件)**:

```
Given ステータスが「処理中」の注文が1件ある
When 管理者がステータスを「発送済み」に変更する
Then 注文のステータスが更新され、顧客にステータス変更通知メールが送信される
```
