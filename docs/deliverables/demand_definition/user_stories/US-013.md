# US-013: 商品を登録する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品管理業務(管理者向け)

**As a** 管理者
**I want** 新しい商品を登録したい
**So that** 顧客に新商品を販売できる

**関連する業務フロー図ステップ**: 商品管理業務 To-Be「商品情報を入力し登録する」

**複雑度判定**: 単純(分岐条件0)

**Confirmation(受け入れ条件)**:

```
Given 管理者が商品名・価格・在庫数を入力する
When 登録する
Then 新しい商品が商品一覧に追加される
```
