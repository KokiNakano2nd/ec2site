# US-003: 注文金額を確認する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 商品購入業務

**As a** 顧客
**I want** 小計・割引・消費税を含めた最終的な支払金額を確認したい
**So that** 納得した金額で注文を確定できる

**関連する業務フロー図ステップ**: 商品購入業務 To-Be「注文金額(小計・割引・税)を計算する」

**複雑度判定**: 単純(分岐条件0: 常に同じ計算ルール)

**Confirmation(受け入れ条件)**:

```
Given カート内商品の小計が10,000円で、割引額が1,000円である
When 注文金額が計算される
Then 消費税は (10,000 - 1,000) × 10% = 900円 として計算される
Then 最終支払金額は 9,900円 として表示される
```
