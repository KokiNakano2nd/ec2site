# US-006: 会員登録を行う

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 会員管理業務

**As a** 顧客(未登録)
**I want** メールアドレスとパスワードでアカウントを登録したい
**So that** カート・注文履歴・お気に入り等の自分専用の情報を扱えるようになる

**関連する業務フロー図ステップ**: 会員管理業務 To-Be「メールアドレス・パスワードを入力し新規登録する」

**複雑度判定**: 単純(分岐条件1つ: メールアドレスが登録済み)

**Confirmation(受け入れ条件)**:

```
Given メールアドレス「a@example.com」が未登録である
When そのメールアドレスとパスワードで新規登録する
Then アカウントが作成され、続けてログイン状態になる

Given メールアドレス「a@example.com」が登録済みである
When そのメールアドレスで新規登録しようとする
Then 「このメールアドレスは既に登録されています」というエラーが表示される
```
