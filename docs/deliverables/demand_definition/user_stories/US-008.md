# US-008: 商品をお気に入り登録・解除する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: お気に入り管理業務

**As a** 顧客
**I want** 気になる商品をお気に入りに登録し、後で見返したい
**So that** 気に入った商品を忘れずに再確認できる

**関連する業務フロー図ステップ**: お気に入り管理業務 To-Be「商品詳細画面でお気に入りボタンを押す」

**複雑度判定**: 単純(分岐条件2つ: 商品が存在しない, 既に登録済み)

**Confirmation(受け入れ条件)**:

```
Given 顧客がログイン済みで、商品Aがまだお気に入りに登録されていない
When 顧客が商品Aのお気に入りボタンを押す
Then 商品Aがお気に入りに追加される

Given 顧客がログイン済みで、商品Aが既にお気に入りに登録されている
When 顧客が商品Aのお気に入りボタンを押す
Then お気に入りが解除される
```
