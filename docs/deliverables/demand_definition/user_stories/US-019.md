# US-019: 売上分析を確認する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 売上分析業務(管理者向け)

**As a** 管理者
**I want** 総売上・日別推移・売れ筋商品・カテゴリ別売上を確認したい
**So that** 経営状況を把握し、仕入れ・販促の意思決定に活かせる

**関連する業務フロー図ステップ**: 売上分析業務 To-Be「売上分析画面を開く」

**複雑度判定**: 単純(分岐条件0: 参照系のみ)

**Confirmation(受け入れ条件)**:

```
Given 注文データが存在する
When 管理者が売上分析画面を開く
Then 総売上・注文数・平均注文額・日別売上推移・売れ筋商品TOP5・カテゴリ別売上が表示される
```
