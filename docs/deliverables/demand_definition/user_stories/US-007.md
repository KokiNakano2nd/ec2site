# US-007: ログインする

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 会員管理業務

**As a** 顧客(登録済み)
**I want** メールアドレスとパスワードでログインしたい
**So that** 自分のカート・注文履歴・お気に入りにアクセスできる

**関連する業務フロー図ステップ**: 会員管理業務 To-Be「メールアドレス・パスワードを入力しログインする」

**複雑度判定**: 単純(分岐条件1つ: メールアドレスまたはパスワードの不一致)

**Confirmation(受け入れ条件)**:

```
Given 正しいメールアドレスとパスワードを入力する
When ログインする
Then アクセストークンが発行され、ログイン状態になる

Given メールアドレスまたはパスワードが間違っている
When ログインしようとする
Then 「メールアドレスまたはパスワードが正しくありません」というエラーが表示される
```
