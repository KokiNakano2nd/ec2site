# US-010: 配送先を登録する

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: 配送先管理業務

**As a** 顧客
**I want** 配送先住所を登録しておきたい
**So that** 注文のたびに住所を入力せずに済む

**関連する業務フロー図ステップ**: 配送先管理業務 To-Be「配送先情報を入力し登録する」

**複雑度判定**: 単純(分岐条件0: デフォルト指定の有無による処理分岐はあるが、顧客からはどちらもエラーなく成功として見える)

**Confirmation(受け入れ条件)**:

```
Given 顧客が配送先を1件も登録していない
When 名前・郵便番号・都道府県・市区町村・番地を入力して登録する
Then 配送先が登録される

Given 顧客が「デフォルトの配送先に設定する」にチェックを入れて登録する
When 登録が完了する
Then 既存の他の配送先のデフォルト指定は解除され、新しい配送先がデフォルトになる
```
