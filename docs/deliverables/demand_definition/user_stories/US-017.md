# US-017: クーポンを有効化・無効化/削除する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: クーポン管理業務(管理者向け)

**As a** 管理者
**I want** 発行済みクーポンを無効化したり削除したりしたい
**So that** キャンペーン終了後の不正利用を防げる

**関連する業務フロー図ステップ**: クーポン管理業務(発行フローと同様のロジックを、既存のクーポンに対して適用する)

**複雑度判定**: 単純(分岐条件1つ: 対象クーポンが存在しない)

**Confirmation(受け入れ条件)**:

```
Given 有効なクーポンが1件ある
When 管理者がそのクーポンを無効化する
Then そのクーポンコードは顧客側で「無効なクーポンコードです」として扱われるようになる
```
