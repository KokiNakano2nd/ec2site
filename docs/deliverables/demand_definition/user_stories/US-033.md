# US-033: クーポン一覧を確認する(管理者)

[← User Story一覧に戻る](../02_user_stories.md)

**業務領域**: クーポン管理業務(管理者向け)

**As a** 管理者

**I want** 発行済みクーポンの状態と利用状況を一覧で確認したい

**So that** 更新・停止・削除すべきクーポンを判断できる

**関連する業務フロー図ステップ**: [[../business_flow/07_coupon_admin|07_coupon_admin.md]]「業務フロー図(To-Be)」

**複雑度判定**: 単純(管理者による作成日時降順の参照のみで状態変更を伴わないため)

**Confirmation(受け入れ条件)**:

```text
Given 管理者がログイン済みで、クーポンが発行されている
When 管理者がクーポン管理を開く
Then 発行済みクーポンが作成日時の新しい順に表示される

Given 一般顧客がログイン済み、または未認証である
When クーポン一覧の取得を試みる
Then 権限不足または認証失敗として拒否される
```
