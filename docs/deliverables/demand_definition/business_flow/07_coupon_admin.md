# 業務フロー図: クーポン管理業務(管理者向け)

[← 業務フロー図一覧に戻る](../01_business_flow.md) / 全体ルール: [[../../../README|docs/README.md]]

### 概要

管理者がクーポン(割引コード)を発行・有効化/無効化・削除する業務。

### 登場アクター

- 管理者
- システム(EC_SITE)

### 業務フロー図(As-Is)

該当なし。業務エキスパートへのヒアリングを行っておらず、既存のクーポン発行方法(紙のクーポン券等)を確認できていないため、憶測でAs-Isを記述することは避け、該当なしとする。

### 課題・問題点

該当なし(As-Is業務を確認できていないため)。

### 業務フロー図(To-Be)

```mermaid
flowchart TD
    subgraph 管理者
        A((●)) --> B["クーポンコード・割引種別・割引値・使用回数上限を入力し発行する<br/><<システム化>>"]
    end
    subgraph システム
        C{"割引種別が percentage または fixed か<br/><<システム化>>"}
        D{"同じクーポンコードが既に存在するか<br/><<システム化>>"}
        E["クーポンを登録する(is_active=true)<br/><<システム化>>"]
    end
    B --> C
    C -->|不正な値| F["「discount_typeは percentage または fixed を指定してください」を表示する"]
    C -->|正しい値| D
    D -->|存在する| G["「このクーポンコードはすでに存在します」を表示する"]
    D -->|存在しない| E
    E --> H((◎))
    F --> H
    G --> H
```

- 有効/無効の切り替え(`PATCH /admin/coupons/{id}`、`is_active`を反転)・削除(`DELETE /admin/coupons/{id}`)は、対象クーポンの存在確認後に更新/削除するだけの単純な処理のため、上図には含めていない。
