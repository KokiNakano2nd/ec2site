# PATCH /addresses/{address_id} — 配送先を編集する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 配送先管理業務 / **リソース**: /addresses

**概要**: 配送先を編集する。**実装上の注記**: バックエンドには実装されているが、フロントエンド(`frontend/src/pages/ProfileView.jsx`)からは呼び出されておらず、UI上の編集機能は提供されていない(F-019参照)
**元になった機能**: F-019
**認証**: 必要(Bearerトークン)

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| address_id | integer(パスパラメータ) | 必須 | 配送先ID |
| (その他) | - | 任意 | `POST /addresses`と同じ項目を、指定したものだけ更新(`exclude_unset`) |

**レスポンス(200)**: 配送先オブジェクト

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 配送先が存在しない、または他ユーザーの配送先 | 「住所が見つかりません」 |
