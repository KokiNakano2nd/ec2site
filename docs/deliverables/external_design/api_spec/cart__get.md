# GET /cart — カート内商品一覧を取得する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /cart

**概要**: カート画面(S-002)の表示データを取得する
**元になった機能**: F-001, F-002
**認証**: 必要(Bearerトークン)

**リクエスト**: なし

**レスポンス(200)**:
| 項目名 | 型 | 説明 |
|---|---|---|
| id | integer | カート行ID |
| quantity | integer | 数量 |
| product | object | 商品情報(`GET /products`と同一形式) |

**エラーレスポンス**: なし
