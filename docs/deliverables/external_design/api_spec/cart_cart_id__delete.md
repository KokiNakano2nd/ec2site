# DELETE /cart/{cart_id} — カートから商品を削除する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /cart

**概要**: カート内商品の削除(×ボタン)
**元になった機能**: F-002
**認証**: 必要(Bearerトークン)

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| cart_id | integer(パスパラメータ) | 必須 | カート行ID |

**レスポンス(204)**: 本文なし

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | カート行が存在しない | 「カート内に該当の商品が見つかりません」 |
