# PATCH /admin/coupons/{coupon_id} — クーポンの有効/無効を切り替える

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: クーポン管理業務(管理者向け) / **リソース**: /admin/coupons

**認証**: 必要(Bearerトークン + 管理者権限)

**概要**: クーポンの`is_active`を反転する(S-102)。リクエストボディで`is_active`を明示指定した場合はその値を設定する(反転ではなく明示上書き)。`low_remaining_uses_threshold`を指定した場合はそのクーポンの残数アラートしきい値を設定・変更する(2026-07-13追加、F-035)
**元になった機能**: F-024, F-035

**リクエスト(任意)**:
| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| is_active | boolean | 任意 | 省略時は現在値を反転する。指定時はその値を設定する |
| low_remaining_uses_threshold | number \| null | 任意 | 残数アラートしきい値。省略時は変更しない |

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | クーポンが存在しない | 「クーポンが見つかりません」 |
