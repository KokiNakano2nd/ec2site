# DELETE /addresses/{address_id} — 配送先を削除する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 配送先管理業務 / **リソース**: /addresses

**概要**: 配送先を削除する(S-007)
**元になった機能**: F-017
**認証**: 必要(Bearerトークン)

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| address_id | integer(パスパラメータ) | 必須 | 配送先ID |

**レスポンス(204)**: 本文なし

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 配送先が存在しない、または他ユーザーの配送先 | 「住所が見つかりません」 |
