# GET /coupons/validate — クーポンコードを検証する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /coupons

**概要**: クーポン適用ボタン押下時に、コードの有効性を確認する(UC-001)
**元になった機能**: F-003
**認証**: 不要

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| code | string(クエリパラメータ) | 必須 | クーポンコード |

**レスポンス(200)**:
| 項目名 | 型 | 説明 |
|---|---|---|
| code | string | クーポンコード |
| discount_type | string | 割引タイプ(`percentage`/`fixed`) |
| discount_value | float | 割引値 |

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 一致する有効なクーポンが存在しない | 「無効なクーポンコードです」 |
| 400 | 使用回数が上限に達している | 「このクーポンは使用回数の上限に達しています」 |
