# GET /config — システム設定を取得する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /payment

**概要**: Stripe決済が利用可能かをフロントエンドが判定するための設定値を取得する(カート画面での「カードで決済」ボタン表示切替に使用)
**元になった機能**: F-005(表示判定のための補助エンドポイント)
**認証**: 不要

**リクエスト**: なし

**レスポンス(200)**:
| 項目名 | 型 | 説明 |
|---|---|---|
| stripe_enabled | boolean | `STRIPE_SECRET_KEY`が設定されているか |

**エラーレスポンス**: なし
