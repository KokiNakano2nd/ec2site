# GET /products/{product_id}/recommendations — おすすめ商品を取得する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /products

**概要**: 指定商品の同一カテゴリを優先し、当該商品を除いた候補から最大4件をランダムに返す(S-001)
**元になった機能**: F-041
**認証**: 不要

リクエスト・レスポンスの型と標準422レスポンスは[OpenAPI正本](../openapi.json)を参照する。

**エンドポイント固有のエラーレスポンス**:

| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 指定商品が存在しない | 「Product not found」 |
