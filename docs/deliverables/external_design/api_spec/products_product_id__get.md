# GET /products/{product_id} — 商品詳細を取得する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /products

**概要**: 商品詳細画面(S-001)の表示データを取得する
**元になった機能**: F-001
**認証**: 不要

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| product_id | integer(パスパラメータ) | 必須 | 商品ID |

**レスポンス(200)**: `GET /products`と同一の商品オブジェクト

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 該当商品が存在しない | `Product not found` |
