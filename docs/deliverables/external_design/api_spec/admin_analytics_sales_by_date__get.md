# GET /admin/analytics/sales-by-date — 日別売上推移を取得する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 売上分析業務(管理者向け) / **リソース**: /admin/analytics

**認証**: 必要(Bearerトークン + 管理者権限)

**概要**: 直近N日の日別売上・注文数を取得する(S-104)
**元になった機能**: F-029

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| days | integer(クエリパラメータ) | 任意(デフォルト30) | 集計対象の日数 |

**レスポンス(200)**: `{date, revenue, count}`の配列

**エラーレスポンス**: なし
