# POST /orders/{order_id}/return-request — 返品を申請する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /orders

**概要**: 発送済み(`shipped`)の自分の注文について、返品理由を添えて返品を申請する(S-004)
**元になった機能**: F-032
**認証**: 必要(Bearerトークン)
**関連する通知**: `04_notification_design.md` N-002

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| order_id | integer(パスパラメータ) | 必須 | 注文ID |
| reason | string | 任意 | 返品理由(自由記述) |

**レスポンス(200)**: 注文オブジェクト(ステータス`return_requested`)

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 注文が存在しない、または当該顧客のものでない | 「注文が見つかりません」 |
| 400 | ステータスが`shipped`以外 | 「この注文は返品を申請できません」 |
