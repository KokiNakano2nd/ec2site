# GET /auth/me — 現在のユーザー情報を取得する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 会員管理業務 / **リソース**: /auth

**概要**: アクセストークンからログイン中ユーザーの情報を取得する(アプリ起動時のログイン状態復元に使用)
**元になった機能**: F-010(付随機能)
**認証**: 必要(Bearerトークン)

**リクエスト**: なし

**レスポンス(200)**: `POST /auth/register`のレスポンスと同一形式

**エラーレスポンス**: なし(トークンが無効な場合は401。この挙動は全認証必須エンドポイント共通のため個別記載は省略)
