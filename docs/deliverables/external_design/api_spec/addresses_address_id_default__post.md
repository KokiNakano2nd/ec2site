# POST /addresses/{address_id}/default — デフォルト配送先を設定する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 配送先管理業務 / **リソース**: /addresses

**概要**: 既存の配送先の中から1件をデフォルトに設定する(S-007)
**元になった機能**: F-016
**認証**: 必要(Bearerトークン)

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| address_id | integer(パスパラメータ) | 必須 | 配送先ID |

**レスポンス(200)**: 配送先オブジェクト(`is_default: true`)

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 配送先が存在しない、または他ユーザーの配送先 | 「住所が見つかりません」 |
