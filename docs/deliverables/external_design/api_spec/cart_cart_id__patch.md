# PATCH /cart/{cart_id} — カート内商品の数量を変更する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /cart

**概要**: カート画面での数量変更(＋／−ボタン)に対応する
**元になった機能**: F-002
**認証**: 必要(Bearerトークン)

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| cart_id | integer(パスパラメータ) | 必須 | カート行ID |
| quantity | integer | 必須 | 変更後の数量 |

**レスポンス(200)**: カート行オブジェクト

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | カート行が存在しない(他ユーザーの行を含む) | 「カート内に該当の商品が見つかりません」 |
| 400 | quantityが1未満 | 「数量は1以上を指定してください」 |
| 400 | quantityが在庫数を超える | 「在庫数を超える数量は指定できません」 |
