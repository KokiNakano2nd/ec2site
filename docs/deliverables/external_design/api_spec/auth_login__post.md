# POST /auth/login — ログインする

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 会員管理業務 / **リソース**: /auth

**概要**: メールアドレス・パスワードで認証し、アクセストークン(JWT)を発行する(S-005)
**元になった機能**: F-010
**認証**: 不要

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| email | string | 必須 | メールアドレス |
| password | string | 必須 | パスワード |

**レスポンス(200)**:
| 項目名 | 型 | 説明 |
|---|---|---|
| access_token | string | アクセストークン(JWT) |

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 401 | メールアドレスが存在しない、またはパスワードが不一致 | 「メールアドレスまたはパスワードが正しくありません」 |
| 429 | 同一IPアドレスから15分間に10回を超えるリクエスト(NFR-022、2026-07-13追加) | 「リクエストが多すぎます。しばらくしてから再度お試しください」 |
