# POST /auth/register — 会員登録する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 会員管理業務 / **リソース**: /auth

**概要**: メールアドレス・パスワードでアカウントを作成する(S-005)
**元になった機能**: F-009
**認証**: 不要

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| email | string | 必須 | メールアドレス |
| password | string | 必須 | パスワード(平文で送信し、サーバー側でハッシュ化) |

**レスポンス(201)**:
| 項目名 | 型 | 説明 |
|---|---|---|
| id | integer | ユーザーID |
| email | string | メールアドレス |
| is_admin | boolean | 管理者フラグ |
| created_at | datetime | 登録日時 |

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 400 | メールアドレスが既に登録済み | 「このメールアドレスは既に登録されています」 |
