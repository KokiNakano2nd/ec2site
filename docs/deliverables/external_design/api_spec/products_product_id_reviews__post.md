# POST /products/{product_id}/reviews — レビューを投稿する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: レビュー投稿業務 / **リソース**: /products/{product_id}/reviews

**概要**: 商品にレビュー(評価・コメント)を投稿する(S-001)(UC-004)
**元になった機能**: F-013
**認証**: 必要(Bearerトークン)

**リクエスト**:
| 項目名 | 型 | 必須 | 説明 |
|---|---|---|---|
| rating | integer | 必須 | 評価(1〜5) |
| comment | string \| null | 任意 | コメント |

**レスポンス(201)**: レビューオブジェクト(`GET /products/{id}/reviews`の要素と同一形式)

**エラーレスポンス**:
| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 商品が存在しない | 「商品が見つかりません」 |
| 400 | 評価が1〜5の範囲外 | 「評価は1〜5で指定してください」 |
| 400 | この顧客がこの商品に投稿済み | 「すでにレビューを投稿済みです」 |
