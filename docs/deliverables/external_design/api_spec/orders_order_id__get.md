# GET /orders/{order_id} — 注文詳細を取得する

[← API仕様一覧に戻る](../02_api_spec.md)

**業務領域**: 商品購入業務 / **リソース**: /orders

**概要**: ログイン中の顧客に属する注文を1件取得する(S-004)
**元になった機能**: F-042
**認証**: 必要(Bearerトークン)

リクエスト・レスポンスの型、共通401/422レスポンスは[OpenAPI正本](../openapi.json)を参照する。

**エンドポイント固有のエラーレスポンス**:

| ステータスコード | 条件 | メッセージ |
|---|---|---|
| 404 | 注文が存在しない、または他の顧客の注文である | 「注文が見つかりません」 |
